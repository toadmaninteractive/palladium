What are the events?

Every game project has a number of events that are triggered when a certain behavior is observed.

Every event has a number of parameters that store the game state.
Ex.: event_name: session_id, account_id, datetime, match_id.. etc

When an event is triggered, parameter values from a particular game client are being sent to the analytics database.
